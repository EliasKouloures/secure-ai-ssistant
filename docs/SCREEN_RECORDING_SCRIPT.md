# 5-Minute Screen Recording Script

1. Start on the top bar and say: "This runs locally and keeps the operator in control."
2. Paste the absence fixture and click `Process locally`.
3. Read the structured facts, internal brief, and subject lines.
4. Open one reply variant and use the copy action.
5. Reset the case.
6. Enter the missing-date phone note and show the clarifying questions.
7. Upload the screenshot fixture and highlight the source provenance.
8. Show the chart, recent-case list, and Diagnostics panel.
9. Close by stating that the repo is public, portable, and ready for extension.
